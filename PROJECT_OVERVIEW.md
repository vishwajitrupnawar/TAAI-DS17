# 📋 Project Overview - Campus Hiring Evaluation

## 🎯 Project Summary

This is a **production-grade notification system** built for the Frontend Track of campus hiring evaluation. It demonstrates full-stack development skills with TypeScript, React/Next.js, Node.js/Express, and Material UI.

### 1. **Logging Middleware** (`logging_middleware/`)
- Reusable Express middleware
- Logs requests, responses, errors
- Console and file logging
- TypeScript implementation
- **Lines of Code**: ~200

### 2. **System Design Document** (`notification_system_design.md`)

- Priority notification algorithm

- API design specifications
- Complexity analysis
- Error handling strategies
- Future improvements roadmap

## 🚀 Getting Started

### Quick Install
```bash
# Run the install script
install.bat

# Or manually:
cd logging_middleware && npm install && cd ..
cd notification_app_be && npm install && cd ..
cd notification_app_fe && npm install && cd ..
```

### Quick Start
```bash
# Terminal 1 - Backend
start-backend.bat

# Terminal 2 - Frontend
start-frontend.bat

# Or manually:
cd notification_app_be && npm run dev
cd notification_app_fe && npm run dev
```

### Access
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
- API: http://localhost:5000/api/notifications
